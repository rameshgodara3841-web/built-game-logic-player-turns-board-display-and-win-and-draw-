Tic Tac Toe (Python)

This is a simple Tic Tac Toe game made using Python.
Two players can play the game in the command line.

Features

- 2 player game (X and O)
- Shows game board
- Detects winner

How to Run

1. Install Python
2. Open terminal or command prompt
3. Run the file:

python main.py

How it Works

- The game uses a list to store the board
- Players take turns (X and O)
- First player to match 3 positions win
